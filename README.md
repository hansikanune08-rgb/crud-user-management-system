# CRUD User Management System

## Project Overview

This project is a CRUD User Management System developed using PHP and MySQL. It allows users to register, log in, update their profiles, and provides administrators with user management capabilities. The project also integrates the OpenWeatherMap API to display live weather information.

---

## Features

- User Registration
- User Login
- Password Hashing
- Role-Based Access Control (Admin/User)
- Admin Dashboard
- Manage Users (CRUD Operations)
- User Profile Management
- Profile Picture Upload
- Weather API Integration
- Session Management
- Form Validation

## Installation

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the project into the htdocs folder.
4. Create a MySQL database named:

user_management

5. Import or create the users table.
6. Update the database credentials in db.php.
7. Open:

http://localhost/CRUD_User_System/register.php
